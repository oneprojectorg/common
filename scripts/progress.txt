## Codebase Patterns
- Button component uses `tv()` (tailwind-variants) for styling with variant props spread through to RACButton
- `cn()` from `../lib/utils` uses `twMerge(clsx(...))` for safe Tailwind class merging/overriding
- LoadingSpinner accepts `className` that gets merged via `cn()` so you can override size/color
- ButtonTooltip extends ButtonProps, so adding props to ButtonProps automatically makes them available on ButtonTooltip
- React Aria `Button` children can be `ReactNode | ((renderProps) => ReactNode)` - handle both cases when wrapping children
